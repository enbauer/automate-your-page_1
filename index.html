Hypertext Markup Language (HTML)-"The Heart of the Web"

HTML is made up of:

  • Text Content (what you see)
  • Markup (what it looks like)
  • References to other documents (e.g. images and videos)

Basic HTML Vocabulary

  • Tag: An HTML tag is always contained within angled brackets. Most tags have an opening tag and a closing tag. Some tags (called "void" tags) do not require a closing tag.
  • Element: An HTML element refers to everything within a set of opening and closing tags.
  • Attribute: This is a property of an HTML element. For example, to set the href attribute of an anchor tag to the Udacity URL, you would write
    a href="www.udacity.com">Udacity URl /a

Inline v Block Elements

HTML elements are usually block-level elements or inline elements. A block-level element occupies the entire space of its parent element, thereby creating a "block". An inline element, however, takes up only as much width as it needs, and does not force new lines.

A block-display element will span the full width of the space available to it, and so will start on a new line in the flow of HTML. The flow will continue on a new line after the block-display element. Some examples of block elements include: div, p, h, ul, ol, and tables.

Inline-display elements donâ€™t break the flow. They just fit in with the flow of the document. Examples of inline elements include: span, a, strong.

Lesson 3:Adding Style to Structure with CSS

What is CSS?

CSS (which stands for Cascading Style Sheets) is a language used to describe the appearance and formatting of your HTML.

A style sheet is a file that describes how an HTML file should look.

We say these style sheets are cascading because the sheets can apply formatting when more than one style applies.

Why Separate Form from Function?

There are two main reasons for separating your form/formatting (CSS) from your functional structure (HTML):

  • You can apply the same formatting to multiple HTML elements without rewriting code over and over again.
  • You can apply similar appearance and formatting to several HTML pages from a single CSS file

Box Models

As discussed earlier, HTML elements are boxes and each box has 4 components:

  • The margin is the space around the element. The larger the margin, the more space between our element and the elements around it. We can adjust the margin to move our HTML elements closer to or farther from each other.
  • The border is the edge of the element. It's what we've been making visible every time we set the border property.
  • The padding is the spacing between the content and the border. We can adjust this value with CSS to move the border closer to or farther from the content.
  • The content is the actual "stuff" in the box. If we're talking about a element, the "stuff" is the text of the paragraph.

Lesson 4:Introduction to "Serious" Programming

Programming

A computer is a machine that can execute a program. With the right program, a computer can do any mechanical computation you can imagine.

A program describes a very precise sequence of steps. Since the computer is just a machine, the program must give the steps in a way that can be executed mechanically. That is, the program can be followed without any thought.

A programming language is a language designed for producing computer programs. A good programming language makes it easy for humans to read and write programs that can be executed by a computer.

Why not write programs in a natural language like English?

Because natural languages are very ambiguous (words/sentences can be interpreted in different ways by different people i.e. the word biweekly)

Because natural languages are very verbose- too many words would be needed to describe something complex and in a precise manner

Getting Started with Python Programming

Python is a programming language. The programs that we write in the Python language will be the input to the Python interpreter, which is a program that runs on the computer. The Python interpreter reads our programs and executes them by following the rules of the Python language.

Lesson 5:Variables and Strings

Variables

A variable is a name that refers to a value (variables give programmers a way to give names to values). In Python, we can use any sequence of letters and numbers and underscores we want to make a variable name, so long as it does not start with a number.

We can use the variable name anywhere we want and it means the same things as the value it refers to.

Strings

A string is a sequence of characters surrounded by quotes; either single or double.

Using Operators on Strings

The + operator between strings will 'add' them together, one after the other.Combining strings together like this is called concatenation. This means that the two concatenated strings are, in essence, pasted together (It is important to note that you cannot concatenate a string and a number).

Indexing Strings

When you want to select sub-sequences from a string, it is called indexing. Use the square brackets[] to specify which part of the string you want to select.

The positions in a string are numbered starting with 0.

When you use negative numbers in the index it starts counting from the back of the string (starting with -1).

Lesson 6:Input-> Function-> Output

What is a Function (procedure)?

A function is a called group of instructuons that return a single result. A function takes input, does something to that input(the called group of instructions), and then produces output. A more detailed summary of what a procedure is and how it is used is explained in the following video:

What is the difference between making and using a function?

Functions are made by starting a line of code with the keyword def and then giving a function name followed by the function parameters in parentheses. These parameters will eventually be replaced by actual values when the function is used (called).

In the "body" of the function, we write the code that specifies what to do with the input parameters. For example the following code could be the definition of a function called square:

def square(x):
   answer = x * x
   return answer

 To use a function, we write the name of the function followed by the value(s)we want to give it in parentheses. Like this:

print square(4)
   >>>16

Lesson 7:Decisions and Repetition: If and While

If Statement

Syntax/grammar

if< test expression>:
         < block>

The block will run if the test expression (comparison) is true. If the test expression does not evaluate to true, then the block doesn't execute.

An if statement ends when there are no more indentations (all of the statements inside the block are indented).

Else Statement

Syntax/grammar

if< test expression>:
         < block>
else:
         < block>

If the test expression is true, the first block will execute. If the test expression does not ealuate to true, the second blocj will execute.

Or Operator

Syntax/grammar

<expression>or<expression>

If the first expression is true, the value is true and the second expression is not evaluated.
If the first expression is false, the value is the value of the second expression.

Loops!

Loops allow things to be done over and over again. A while loop, for example, will continue to run as long as the test expression is true. Once it is false, we go to the next expression.

Syntax/grammar

while< test expression>:
         < block>

A break statement gives us a way to stop a loop even if the test condition is still true.

Lesson 8:Structured Data- Lists and For Loops

String v List

A string is a sequence of characters, whereas a list is a sequence of anything (characters, strings, numbers, other lists, etc.) Lists are very versatile!

When storing a list, use square brackets, and separate the elements of the list with commas.ex:
<list>-->[<expression>,<expression>,...]
A list with zero elements is known as an empty list and looks like this:
<list>-->[]

Nested Lists

List can be nested (lists inside lists)

Ex. Beatles=[['john', 1940], ['paul', 1942], ['george', 1943],['ringo', 1940]]

If we call 'Beatles', it will print all of our lists. If we call 'Beatles[3]' it will print the fourth position list. If we call 'Beatles[3][1]' it will print the second position value of the fourth position list.

Mutation and Aliasing

Lists support mutation. Mutating a list means that we can change the value of a list after we've created it. In other words, mutation modifies an existing object. Strings that cannot be mutated are imutable.

When we assign a different variable name to an existing list, that is aliasing (when we have two different ways of referring to the same object).

List Operations

append
<list>.append(<element>)

Append mutates the list that it is invoked on (it's not creating a new list).We pass in the element we want to append to the end of a list.

+
<list>+<list>

The + operation produces a new list by adding the contents of two lists together (it doesn't mutate either list).

len
len(<list>)

We pass into len the list we want to know the lenfth of. The output is the number of elements in the input.

Loops on Lists

for loop
A for loop is used when you have a piece of code you want to repeat n number of times. For a detailed explanation with examples, check out this great description.

index
<list>.index(<value>)

The index method is invoked on a list. You pass in a value, and the output of the index is the postutuon that the value exists in the list. If the list doesn't contain the value we passed in, index returns an error (not -1).

in
<value>in<list>

Simple and straightforward. If the specified value is in the list, the output is true. Otherwise, the output is false. Similarly, <value>not in<list> has the opposite meaning.
